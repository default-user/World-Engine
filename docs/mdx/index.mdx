# World Engine Documentation

Welcome to the World Engine documentation. World Engine is a desktop-first, persistent world
space engine with VR as an optional embodiment mode.

## Architecture at a Glance

One world kernel, two embodiment modes. Desktop is primary. VR is a second input and camera
modality that shares the same world truth, persistence, and authoring operations.
